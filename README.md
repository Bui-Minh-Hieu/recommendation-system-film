# Movie Recommendation System - Graduation Thesis

### Introduction
This is the source code for my graduation thesis at HAUI.
The project builds a movie recommendation website using Django and Machine Learning (Collaborative Filtering).

### Technologies
- Python / Django
- Scikit-learn (Machine Learning)
- PostgreSQL
- TMDB API (for movie metadata)
